# Sabarinathan Karuppusamy — Portfolio

Personal portfolio website for [sabari2001.github.io](https://sabari2001.github.io/) showcasing AI/ML engineering work, projects, and professional experience.

## Features

- Modern, responsive design with glassmorphism navigation
- Dark / light mode toggle
- Interactive project filtering
- Experience timeline
- Animated skill bars and typewriter hero
- Particle.js background
- Google Analytics integration
- SEO meta tags

## Tech Stack

- HTML5, CSS3 (Grid, Flexbox, CSS Variables)
- Vanilla JavaScript
- Font Awesome 6
- Google Fonts (Sora + Inter)
- Particles.js

## Local Development

```bash
git clone https://github.com/SABARI2001/SABARI2001.github.io.git
cd SABARI2001.github.io
# Open index.html in a browser, or use a local server:
python -m http.server 8080
```

## Deployment

This repo is configured for **GitHub Pages**. Push to `main` and enable Pages in repository settings.
